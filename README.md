## Overview 🌟
Welcome to the **Python for AI Developers** repository — a curated collection of Jupyter notebooks and learning resources designed to bring Java-experienced developers up to speed with Pythonic programming. This repository serves as a comprehensive recap of core Python concepts, object-oriented design, and functional programming patterns, with practical examples and hands-on exercises throughout.

## About the Repository 📖
This repository takes a **Java → Python** bridging approach, making it ideal for developers who already have programming experience and want to master Python quickly and effectively. Each module is structured as a progressive series of interactive notebooks that emphasize real-world patterns and best practices.

## Getting Started 🚀
To get started with Python for AI Developers:

1. Ensure **Python 3.10+** is installed on your machine.
2. Clone the repository:
   ```bash
   git clone https://github.com/pmoschos/ai-for-developers
   ```
3. Navigate to the `ai-for-developers` directory.
4. Open any notebook in **Jupyter** or **VS Code** with the Jupyter extension.
5. Follow the notebooks in order (01 → last) within each module.
